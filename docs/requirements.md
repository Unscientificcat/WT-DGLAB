# 项目需求规格

## 项目概述

**WT-DGLAB** 是一款 Windows 桌面软件，将战争雷霆（War Thunder）游戏数据与郊狼 3.0（Coyote 3.0）电击设备联动，实现游戏数据实时驱动电击反馈。

## 核心需求

### R1 — 飞机过载电击联动
- 读取战争雷霆实时过载数据（G-force / `ny` 字段）
- 用户配置过载下限（如 1G）→ 电击强度 0
- 用户配置过载上限（如 10G）→ 电击强度最大
- 中间值线性映射为电击强度

### R2 — 坦克损伤电击联动
- 读取战争雷霆坦克损伤数据（损毁部位数量 / 部件血量）
- 用户配置损伤下限 → 电击强度 0
- 用户配置损伤上限 → 电击强度最大
- 中间值线性映射为电击强度

### R3 — 用户交互界面
- 连接状态显示（WT 状态、郊狼状态）
- 实时数据面板（当前过载/损伤数值、电击强度进度条）
- 参数设置面板（上下限、通道强度等）
- 手动切换飞机/坦克模式
- QR 码显示区（手机扫码连接）
- 淡蓝色主题，简洁直观

### R4 — 双通道支持
- 支持郊狼 A/B 两个通道独立设置最大强度
- 两个通道同时输出不同的电击强度
  - A 通道强度 = 主映射值
  - B 通道强度 = 主映射值 × (B_max / A_max)

### R5 — 参数持久化
- 所有用户设置保存为 JSON 文件
- 软件启动时自动加载上次配置
- 支持一键恢复默认设置
