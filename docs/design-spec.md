# 设计规范

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F0F6FC` | 极淡蓝白 |
| 卡片/面板 | `#FFFFFF` | 白色 |
| 主色调 | `#4A90D9` | 淡蓝，按钮/进度条 |
| 悬停 | `#357ABD` | 深蓝，按钮悬停 |
| 强调 | `#5BA0E8` | 亮蓝 |
| 文字主色 | `#2C3E50` | 深灰蓝 |
| 文字次要 | `#7F8C8D` | 灰色 |
| 成功 | `#27AE60` | 绿色，已连接 |
| 错误 | `#E74C3C` | 红色，未连接 |
| 边框 | `#D6E4F0` | 淡蓝灰 |
| 输入框背景 | `#F7FAFD` | 极淡蓝 |

## UI 布局

窗口尺寸：960×700（最小 900×650）

```
┌──────────────────────────────────────────┐
│ [状态栏] WT连接 ●  郊狼连接 ●  地址      │
├──────────────┬───────────────────────────┤
│ 实时数据面板  │ 设置面板                  │
│ (固定280px)  │ - 模式选择(飞机/坦克)      │
│              │ - 飞机设置(上下限/通道)    │
│  🛩 图标     │ - 坦克设置(上下限/通道)    │
│  当前过载值  │ - 连接设置(端口/间隔)      │
│  进度条      │ - 保存/恢复按钮            │
│  A通道强度   │                           │
│  B通道强度   │                           │
├──────────────┴───────────────────────────┤
│ [QR码] 扫码连接  提示文字                │
└──────────────────────────────────────────┘
```

## 字体

- 主字体：Microsoft YaHei / Segoe UI / PingFang SC
- 数值大字：28px bold
- 正文：13px
- 提示文字：11-12px

## 交互反馈

- 保存按钮：点击后短暂变绿并显示"✓ 已保存"(1.5s)
- 状态指示灯：绿色圆点 = 已连接，红色圆点 = 未连接
- 进度条：实时反映当前电击强度百分比
